# 电力交易升级流程（绿电交易全链路）

## 跨系统联动（7大系统）

### 系统架构
| 系统 | 核心职责 |
|------|---------|
| **VPP（绿电门户）** | 主数据维护、交易导入、财务审核、盈亏损益处理 |
| **CBS系统** | 采购单管理、发票校验、请款申请 |
| **SAP** | 收入记账、成本记账、暂估/冲销 |
| **票税通** | 发票开具 |
| **UPP** | 发票核销记账 |
| **云报账** | 审核出款 |
| **影像系统** | 附件存储归档 |

### 主数据管理
- 零售用户：88码
- 渠道代理商：V码
- 交易代理商：V码

### 核心流程
1. **VPP端操作**
   - 登录VPP，进入绿电门户
   - 维护主数据信息（零售用户/渠道代理商/交易代理商）
   - 导入零售用户报退明细数据（校验代理商/渠道商系统是否存在）
   - 进入【电力交易】菜单导入电网交易数据（自动按省份+月份匹配渠道商明细）
   - 上传结算单等附件，提交审核
   - 财务审核（通过/驳回重新导入）

2. **盈亏损益处理**
   - **盈利场景**
     - 国网：按单据给国网客户上收入 → 传SAP上收入（借：应收账款 客户88码 / 贷：主营业务收入 6001010201）
     - 渠道商：按单据生成采购单，暂不记成本 → CBS同步
     - 交易代理商：按单据记暂估成本（按导入的代理商分成金额记暂估）→ 传SAP记暂估（借：主营业务成本 6401010201 / 贷：应付账款 2202030000）
   - **亏损场景**
     - 国网：按单据给国网记成本生成采购单，冲对应收入记反向分录（借：主营业务成本 客户编码 / 贷：主营业务收入 6001010201）
     - 渠道商：按单据生成采购单，暂不记账 → CBS同步

3. **CBS系统流程**
   - 生成采购单 → 同步至CBS-采购单管理
   - 发票校验：上传发票信息传票进行校验，状态「已占用」
   - 请款申请：填写请款信息，选择发票、采购单，上传附件提交
   - 发票后到类型：请款后在【发票校验】上传发票附件进行票验
   - 选择发票传UPP核销 → 附件传影像系统

4. **SAP记账规则**
   - 上收入：借 应收账款 客户88码 / 贷 主营业务收入 6001010201
   - 记暂估：借 主营业务成本 6401010201 / 贷 应付账款 2202030000
   - 冲负向收入：借 主营业务成本 客户编码 / 贷 主营业务收入 6001010201
   - 记成本：借 主营业务成本 6401010201 / 贷 应付账款 2202030000

### 交易代理商年度结算子流程
1. 新增【交易代理商年度结算】菜单
2. 导入每个交易代理商每年的结算成本数据（代理商名称、V码、年份、交易总金额）
3. 提交财务审核（审核时显示该代理商当年累计的暂估总金额）
4. 审核通过 → 生成采购单，暂不记成本
5. 根据代理商V码匹配【电力交易】中该代理商年度暂估总金额
6. 传SAP冲一笔全年暂估（借 主营业务成本 6401010201 / 贷 应付账款 2202030000）
7. 同步到CBS-采购单管理 → 发票校验 → 请款申请 → UPP核销 → 云报账出款 → 影像系统归档

### 新增商品类型
- 渠道商代理商：「售电服务费」
- 国网：「售电结算损益」

### 关键交互点
- VPP → CBS：采购单同步
- VPP → SAP：收入/暂估/冲销/成本记账
- CBS → UPP：发票核销记账
- CBS → 影像系统：附件存储
- UPP → 云报账：审核出款
- SAP ← VPP：返回记账凭证号
