# 光伏服务商入驻 · 新人培训手册

> 面向业务与产品经理 | 2026-06-08 整理  
> 来源：业务讲解（薛荣基）+ 知识库 + 源码验证  
> 本文档用于新人快速理解"服务商入驻"业务的来龙去脉

---

## 一、为什么需要服务商入驻？

**一句话**：海尔新能源靠电站盈利，电站不能从地里长出来，需要服务商帮建。服务商入驻是建站流程的**前置准备**，所有后续流程都依赖入驻环节打通的数据。

### 服务商是谁？

- **来源**：市场部招商招进来，不是自己注册进来的
- **角色**：替海尔在全国各地找屋顶、建电站、安装施工
- **收入**：建一个电站约 2-3 万元（视功率而定）
- **系统入口**：商户通（CBS 商务通），不是鸿蒙系统

---

## 二、入驻全流程总览

```
招商招引 → 注册商务通账号 → 申请入驻 → 分中心初审 → 总部审核
    ↓
电签认证 → 签署主合同 → 启用仓库 → 启用区域 → 缴纳保证金 → 发货额度生效
    ↓
可以开始登记电站、下单配货、建设电站
```

> **核心原则**：前面任何一步没走完，后续建站就会卡住。

---

## 三、每一步详细说明

### Step 1：申请入驻

#### 1.1 入口
服务商登录商户通 → 选择「**新能源光伏管理**」（数字建站），这是光伏正确的入驻入口。页面上还有其他入口（合伙人、EPC等），是其他业务线用的。

#### 1.2 填写信息

| 字段 | 说明 | 为什么重要 |
|---|---|---|
| **营业执照信息** | 公司名称、税号等 | 基础核验 |
| **所属分中心** | 合肥一、合肥二、江苏一等 | 决定哪个分中心参与初审；分中心才能看到该服务商 |
| **服务区域** | 服务商自填的服务区域 | 审核参考，告知审核人他想在哪干 |
| **仓库地址** | 服务商主仓库地址 | 后续 SAP 仓库管理的初始地址，发货要用的 |
| **被授权人姓名/电话** | 电签运营责任人 | 影响后续合同电签的责任人是谁 |

### Step 2：分中心初审

#### 2.1 MDM 主数据校验（关键前置条件）

- **服务商公司必须提前在集团 MDM 主数据平台建户**
- 系统用**税号**去 MDM 实时查询，获取：
  - **V 码（微码）**：供应商编码——海尔给服务商**付钱**用
  - **8 码**：客户编码——服务商给海尔**付钱**用
- 如果 MDM 查不到 → 提示"MDM 主数据不存在"→ 审核无法进行
- **常见坑**：MDM 用英文括号，CBS 用中文括号 → 匹配不上

#### 2.2 审核人操作

- **关联公司**：一定是 **EP吉林**（青岛海尔光伏新能源）
- **模式选择**：互用 EPC、A02、H01 等 → 决定服务商替我们建什么类型的电站（不同资方）
- **授权区域**：设置该服务商允许建电站的地理范围（省→市→区）
  - 授权区域 = 电站登记时能选择的地址
  - 后续可新增申请更多区域
- **月度完成功率**：填的数字会出现在主合同中（总和作为合同指标）

#### 2.3 税务登记号 = 第二张身份证

> ⚠️ **极重要**：税号填错后续全错
> - 后续合同签署、电签认证、MDM 查询都依赖税号
> - 一旦填错，可能以别的公司身份签合同，后果严重
> - 审核时必须对照营业执照认真核对

### Step 3：总部审核

分中心审核通过后，进入总部审核。总部通过后，系统授予服务商**商务通服务商角色**——相当于开通了使用系统功能的"钥匙卡"。

此时服务商登录后能看到「新能源管理」模块。

### Step 4：电签认证 & 签署主合同

#### 4.1 DCC 电签平台
- 服务商需要在电签平台完成注册和认证
- 认证身份 = 入驻时填的税号对应的公司
- ⚠️ 如果税号填错，会认证成别的公司

#### 4.2 签署主合同（SP 合同）
- 合同编号以 **SP** 开头（Service Provider）
- 甲方：EP吉林，乙方：服务商
- 合同内容包含：授权区域、月度完成功率、联系人、邮箱等（都来自入驻时填的数据）
- 电签盖章后，状态变为**运营中**

### Step 5：仓库启用

**为什么需要仓库？** 海尔要给服务商发货（组件等物料），服务商要收货，需要有仓库地址。

- 仓库初始地址 = 入驻时填写的仓库地址
- 仓库在系统中初始是**待启用状态**
- **SAP 库位编码**：必须由供应链人员在 SAP 系统中单独维护
  - 每个仓库对应唯一库位码
  - 所有入库、出库、调拨等库存操作都依赖此码
  - 不配 → 配货时选不到仓库 → 无法出库
- 审核人员和供应链是两批人 → 所以不是审核通过自动生成的

### Step 6：授权区域启用

- 审核时分配的授权区域 = 初始默认区域
- 区域审核通过后才能**启用**
- 区域未启用 → 电站登记时选不到这个地址 → 建不了电站
- 服务商可申请新增区域（如原本授权深圳，后来想去珠海干）

### Step 7：保证金（押金）

- 海尔需要向服务商收取保证金，防止"空手套白狼"
- 保证金交完后 → CBS 保证金管理可见
- 需要经过**对账系统**和**银行流水比对**后才能生效
- 交完保证金 → 获得**发货额度**

### Step 8：发货额度 & 配货

| 额度类型 | 说明 |
|---|---|
| **可发货额度** | 可用于申请发货的组件功率上限 |
| **已发货额度** | 已发给服务商的货占用的额度 |

**额度流转**：
1. 申请配货 → 占用可发货额度
2. 确认入库 → 扣减额度
3. 电站施工完成（物料安装到电站上） → 释放额度回到可发货额度
4. 额度为空 → 不能再发货

## 四、入驻完成后的建站流程衔接

入驻走完后，服务商才能：
- ✅ 电站登记时选到自己的授权地址
- ✅ 下单配货时选到自己的仓库
- ✅ CBS 供应链根据额度给他配货
- ✅ 签署各类合同（主合同、项目协议、供电合同等）

**缺少任何前置条件都会卡住**：
- 仓库没启用 → 配货时选不到仓库
- 区域没启用 → 登记时选不到地址
- 保证金没交 → 没有发货额度，无法出库
- 税号填错 → 认证成别的公司，合同签署错误

## 五、关键业务概念速查

| 概念 | 解释 |
|---|---|
| **MDM** | 集团主数据平台，维护企业客户/供应商信息 |
| **V 码（微码）** | MDM 供应商编码，海尔给服务商付钱的依据 |
| **8 码** | MDM 客户编码，服务商给海尔付钱的依据 |
| **税号** | 服务商在系统中的"第二张身份证"，贯穿全流程 |
| **EP吉林** | 青岛海尔光伏新能源，服务商关联的唯一公司 |
| **分中心** | 海尔在全国各地的分支机构，负责初审和区域管理 |
| **SAP 库位码** | 供应链系统中仓库的唯一标识，库存操作的基础 |
| **DCC** | 电子签章平台，合同签署走这个系统 |
| **商户通** | 服务商使用的系统（商务通），区别于 CBS 管理后台 |
| **CBS** | Central Business System，海尔内部管理后台 |

## 六、涉及的核心数据表

| 表名 | 数据库 | 说明 |
|---|---|---|
| `corporate_client` | rrsjk_light | 企业客户主表（状态、商户ID、保证金等） |
| `merchant` | rrsjk_light | 商户表（税号、客户编码、类型、状态） |
| `merchant_extras` | rrsjk_light | 商户扩展信息（地址、营业执照、银行信息等） |
| `merchant_region` | rrsjk_light | 商户授权区域（MASTER 类型） |
| `corporate_client_deposit` | rrsjk_light | 零碳适家企业客户押金/保证金 |
| `light_station` | rrsjk_light | 电站主表（163 个 Mapper 引用，最核心的表） |

## 七、常见线上问题

1. **MDM 查不到数据**：MDM 用英文括号，CBS 用中文括号 → 税号匹配不上
2. **税号填错**：后续电签认证、合同签署全部关联错误的公司
3. **仓库没配 SAP 库位码**：配货时选不到仓库，无法出库
4. **区域没审核启用**：电站登记选不到地址，建不了电站
5. **保证金未对账生效**：额度不足，无法发货

---

## 八、分中心与服务商的关系

- 分中心分布全国各地，管理辖区内的服务商
- 服务商入驻时选择所属分中心 → 该分中心参与初审
- 分中心需要手机操作 → CBS 不支持移动端 → 正在开发**分中心 APP**
- 分中心 APP 本质是把 CBS 的核心功能做移动端适配

---

## 九、后续流程

入驻完成后，服务商进入：
1. **电站登记** → 选择地址、录入业主信息
2. **方案审核** → 分中心/总部审核方案和配置
3. **下单配货** → 根据额度申请物料
4. **施工建设** → 服务商安装施工
5. **技术审核** → 审核施工质量和并网资料
6. **商务审核** → 审核合同签署、并网信息
7. **并网发电** → 连续三天发电，正式并网
