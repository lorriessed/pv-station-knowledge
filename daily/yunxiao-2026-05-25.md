## 业务链分析

### 1. TAEI-3073 广发并网进件 — 状态机修复（代码明确证明）

**提交**: majinhu `52621ae` (2026-05-22) + 多个相关提交
**分支**: `origin/20260508-guangfa-binwang`
**涉及文件**:
- `GfCompleteInputPieceProcess.java` — 完工进件推送流程
- `GfMergeGridInputPieceProcess.java` — 并网进件推送流程

**变更内容**:
- **完工进件推送失败**: 退回至 `TECH_CHECK_REJECT`（技术审核驳回）
  ```
  技术审核通过 → 推完工进件 → [失败] → 退回技术审核驳回
  ```
- **并网进件推送失败**: 退回至 `FIRST_AUDIT_REJECT`（商务审批驳回）
  ```
  商务验收通过 → 推并网进件 → [失败] → 退回商务审批驳回
  ```
- 完整流程: `技术审核通过 → 推完工进件 → 广发完工审核通过 → 上传并网资料 → 商务验收通过 → 推并网进件`

**业务意义**: 修复了推送失败后电站状态未正确回退的问题，确保异常情况下电站不会卡在中间状态。
