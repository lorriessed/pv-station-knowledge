## 核心业务链分析

### 1. TAEI-3103 — 电站业主租金停付管理 (完整业务链)

**业务背景**: 业主因纠纷等原因申请暂停租金支付，需通过分中心初审→总部终审两级审核流程，后续可申请恢复。

**完整业务链**:
```
nahui-pv.hds-h5 (李培龙) → 前端页面: 停付申请列表/详情/审核
    ↓ Dubbo
rrsjk-hds-web (sunzn) → RentSuspendApplicationController
    - 初审通过/驳回接口
    - 总部终核通过/驳回接口
    - 业主租金恢复申请及审核接口
    ↓ Dubbo
rrsjk-light-service (sunzn) → RentSuspendApplicationServiceImpl
    - 申请单 CRUD + 状态流转
    - 关联租金计划记录 (rent_suspend_application_record)
    - 扣除月数计算
    - 工单状态验证 (只有已关闭工单才能恢复)
    ↓ MyBatis
light_unionpay_rent_record (新增 SUSPEND 支付状态)
rent_suspend_application (申请表)
rent_suspend_application_record (关联记录表)
```

**状态机** (10 种状态):
```
SUSPEND_WAIT_FIRST_AUDIT → SUSPEND_FIRST_AUDIT_REJECTED (驳回)
                         → SUSPEND_WAIT_FINAL_AUDIT → SUSPEND_FINAL_AUDIT_REJECTED (驳回)
                                                   → SUSPEND_FINAL_AUDIT_APPROVED (停付生效)
                         → SUSPEND_CANCELED (撤销)

RESUME_WAIT_AUDIT → RESUME_AUDIT_REJECTED (可重新提交)
                  → RESUME_AUDIT_APPROVED (恢复生效)
```

**代码明确证明**: `RentSuspendApplication.java` StatusEnum 10 种状态 (commit: c71e58c5c4)
