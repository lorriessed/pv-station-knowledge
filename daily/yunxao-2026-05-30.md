## 业务链分析

### 1. TAEI-3159: 放开产值确认法的收入闸口
- **业务语义**: 工商业光伏项目使用产值确认法计算收入时，原来限制收入金额不能超过产值设计值（材料收入、施工服务收入、设计收入三项校验），现在放开此限制
- **核心变更**: `CmProductionValueIncomeServiceImpl` 中三项收入校验被注释掉，成本校验保留
- **调用链**: 产值确认提交 → `CmProductionValueIncomeServiceImpl` → 校验收入/成本 → SAP记账
- **影响**: 允许工商业项目超额上报收入，需关注财务数据一致性

### 2. TAEI-3107: 风电产值法兼容1.0
- **业务语义**: 工商业风电项目的收款里程碑不执行预付款金额校验（与光伏项目区分）
- **字段重构**: 工商业领用报表从4个独立比例字段合并为 `nodeFinishPct`（节点完成比例）
- **调用链**: 收款里程碑导入 → `CmLightProjectServiceImpl` → 判断业务类型(WIND_POWER) → 跳过预付校验
- **涉及表**: `cm_light_use` (nodeFinishPct字段)、`cm_construction_progress_audit`
- **影响**: 风电项目收款流程与光伏项目分离

### 3. TAEI-3092: 农户租金类个税报表及记账
- **业务语义**: 新增租金个税计算、报表展示、SAP记账功能
- **计算逻辑**: 个税 = (月租金 - 800免税额) × 20% (月租金 ≤ 4000) 或 月租金 × 0.8 × 20% (月租金 > 4000)
- **SAP记账**: 记账状态追踪 (WAIT/SUCCESS/FAIL)，失败原因记录，支持重试
- **调用链**: 租金数据 → `RentTaxAmountServiceImpl` → 计算个税 → SAP记账 → 更新状态
- **前端**: 明细列表 + 合计列表 + 批量确认记账按钮
- **状态**: 测试中

### 4. TAEI-3146: 发电数据取数逻辑切换大数据平台
- **架构变更**: 从 DWS 层迁移到 ADS 层，新增独立 ADS 数据源
- **数据源**: `spring.ads.datasource` 与原有 DWS 数据源并行
- **表映射**: 6张 ADS 报表表替换原 DWS 表（无 id 字段）
- **调用链**: 前端查询 → `LightInveterController` → ADS DAO → ADS 表
- **字段修正**: `inveter_sn` → `inverter_sn` (拼写修正)
- **状态**: 开发中
