## 关键业务变更

### 1. [重要] 方案审核通过不再强制要求逆变器配置
- **来源**: `rrsjk-admin-bff` → `LightStationController.java` (yumiao, commit a4bbb3a3, 2026-06-25)
- **旧逻辑**: `auditPass()` 方法在 `configList` 为空时直接抛异常 "审核通过需要填写逆变器配置"
- **新逻辑**: `configList` 变为可选参数，为空时不设置到 `serviceRequest` 中，直接调用 `lightStationClient.auditInfoOk()`
- **业务含义**: 方案审核通过时可以先不配置逆变器，后续再补充。降低了审核通过的门槛。

### 2. 电站列表新增"三天发电时间"查询条件
- **来源**: `rrsjk-admin-web` → `LightStationController.java` (wangxiran, commit 27506189, 2026-06-25)
- **变更**: `doListRefactor.do` 和 `doExport.do` 新增 `firstThreePowerAtStart/End` 参数
- **业务含义**: 支持按"首次三天发电时间"范围筛选电站列表和导出

### 3. 技术审核人员新增删除操作
- **来源**: `rrsjk-admin-web` → `LightTechAuditorController.java` (baoxin, commit 8554d2f7, 2026-06-24)
- **新增**: `delete.do` 端点，权限 `tech:auditor:delete`
- **实现**: 软删除，设置 `status = DELETED`，复用 `editAuditorStatus()` 方法

### 4. 逆变器图片审核驳回功能
- **来源**: `rrsjk-admin-web` → `LightStationSkillController.java`, `ReviewMaterialController.java` (wangxiran, commit 14044db7, 2026-06-24)
- **新增**: 技术审核中支持对逆变器图片进行单张驳回
- **关联**: `LightInverterShowVo` 新增 20 行代码（新增字段）

### 5. 商务验收审核逆变器列表切换为 DWS
- **来源**: `rrsjk-admin-web` → `lightStationBusinessAudit.ftl` (majinhu, commit aa467f1e, 2026-06-25)
- **变更**: 商务验收审核页面的逆变器列表数据源切换为 DWS

### 6. 租金个税统计增强 (TAEI-3255)
- **来源**: `rrsjk-admin-web` → `RentTaxRecordController.java`, `RentTaxAmountRecordExcel.java`, `RentTaxAmountSummaryExcel.java` (代继宁, commits f10d06f1/0ef268c8/905b551e, 2026-06-23)
- **新增字段**: 手机号、记账状态、所属公司
- **业务逻辑**: 根据所属公司区分海尔/金泰记账接口调用
- **导出增强**: 明细和汇总均支持新字段查询和导出

### 7. 零碳套餐库存扣减类型扩展 (TAEI-3144)
- **来源**: `rrsjk-item-service` → `ZeroCarbonItemSetMealStockServiceImpl.java` (代继宁, commits 37d98614/1f1d160b, 2026-06-17~18)
- **变更**:

### 8. HDS 运维电站导出性能优化
- **来源**: `rrsjk-hds-web` → `HdsLightStationController.java` (sunzn, commit 9b194afe, 2026-06-25)
- **优化**:
